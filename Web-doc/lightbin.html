📦 LightBin Light-Scan Binary Packet Format

v1.0 Fast Loading Compact Storage zlib Compression Light-Scan's native binary packet format

Overview

LightBin is a custom binary packet format designed for efficient storage and retrieval of captured network packets. It is used by LightSniff and LightLab to save packet captures in a compact, fast-loading format.

⚡ Fast Loading

LightBin files load 1.2–2× faster than PCAP because packets are stored with minimal parsing overhead.

📦 Compact Storage

LightBin files are typically 1–6% larger than PCAP, but the speed and metadata gains justify the trade-off.

📊 Rich Metadata

Stores capture arguments, statistics, tool version, packet types, and per-packet timestamps — all inside the file.

🔄 Cross-Platform

Designed to be portable across systems with explicit endianness and consistent data types. Works on Windows, Linux, macOS, and BSD.

🗜️ zlib Compression

Full zlib compression for both packet data and metadata. Reduces file size significantly for large captures.

🔍 Smart Packet Detection

Auto-detects packet types (Ethernet, VLAN, IPv4, IPv6, ARP, raw) and stores packet_types in metadata for faster analysis.

File Structure

Header (24 bytes)

Offset Size Field Description
0–3 4 Magic LBN\x00 — identifies LightBin format
4–7 4 Version Format version (currently 1)
8–11 4 Created Unix timestamp (creation time)
12–15 4 Packet Count Total number of packets in the file
16–19 4 Flags Bitmask: 0x01 (Compressed), 0x02 (Metadata)
20–23 4 Checksum CRC32 of version, created, count, and flags

Packet Data Section

Each packet is stored sequentially. Packets are not compressed individually; compression is applied to the entire file or not at all.

Packet Header (12 bytes)

Offset Size Field Description
0–7 8 Timestamp Packet capture time (double precision float, Unix epoch)
8–11 4 Size Packet size in bytes (unsigned integer)

Packet Data (Size bytes)

Raw packet bytes as captured from the network.

Metadata Section

The metadata section is always present. If FLAG_COMPRESSED is set, the entire metadata block is compressed with zlib.

Metadata Size (4 bytes)

Length of the metadata block in bytes.

Metadata Data (Variable)

Serialized metadata using Python's pickle format containing:

  • version — LightBin version used
  • created — Creation timestamp
  • packet_count — Number of packets
  • args — Capture arguments (interface, filter, etc.)
  • stats — Capture statistics
  • tool — Tool that created the file (e.g., "LightSniff v1.0.1")
  • packet_types — List of packet types detected
  • packet_timestamps — List of packet timestamps

Complete File Structure Diagram

+---------------------------+ | LBN\x00 (4 bytes) | Magic Header +---------------------------+ | Version (4 bytes) | LightBin Version (1) +---------------------------+ | Created (4 bytes) | Unix Timestamp +---------------------------+ | Packet Count (4 bytes) | Number of Packets | (Updated at end of write) | +---------------------------+ | Flags (4 bytes) | Bitmask of Features | (COMPRESSED | METADATA) | +---------------------------+ | Checksum (4 bytes) | CRC32 Integrity +---------------------------+ | | Packet 1 Header | +-------------------+ | | Timestamp (8) | | | Size (4) | | +-------------------+ | Packet 1 Data | (Size bytes) | | Packet 2 Header | +-------------------+ | | Timestamp (8) | | | Size (4) | | +-------------------+ | Packet 2 Data | (Size bytes) | | ... | | Metadata Section | +-------------------+ | | Metadata Size (4) | | +-------------------+ | Metadata Data | (Serialized dict) +---------------------------+

Implementation (LightSniff)

Constants

            import struct
            import pickle
            import time
            import zlib

            LIGHTBIN_MAGIC = b'LBN\x00'
            LIGHTBIN_VERSION = 1

            FLAG_COMPRESSED = 0x01
            FLAG_METADATA = 0x02

Saving to LightBin

            def save_binary(filename, packets, compress, args=None, stats=None):
                try:
                    creation_time = int(time.time())
                    packet_count = 0
                    packet_types = []

                    FLAG = FLAG_COMPRESSED if compress else FLAG_METADATA

                    with open(filename, 'wb') as f:
                        header = struct.pack(
                            '<4sIIIII',
                            LIGHTBIN_MAGIC,
                            LIGHTBIN_VERSION,
                            creation_time,
                            0,
                            FLAG,
                            0
                        )
                        f.write(header)
                        header_pos = f.tell() - 24

                        for pkt in packets:
                            timestamp = time.time()
                            raw_bytes = bytes(pkt)

                            if pkt.haslayer(Ether):
                                if pkt[Ether].type == 0x8100:
                                    packet_types.append('ether-vlan')
                                packet_types.append('ether')
                            elif pkt.haslayer(IP):
                                packet_types.append('ip')
                            elif pkt.haslayer(IPv6):
                                packet_types.append('ipv6')
                            else:
                                packet_types.append('raw')

                            f.write(struct.pack(' < dI ', timestamp, len(raw_bytes)))
                            f.write(raw_bytes)
                            packet_count += 1

                        metadata = {
                            'version': LIGHTBIN_VERSION,
                            'created': creation_time,
                            'packet_count': packet_count,
                            'args': vars(args) if args else None,
                            'stats': stats,
                            'tool': f'LightSniff v{Version}',
                            'packet_types': packet_types
                        }

                        metadata_bytes = pickle.dumps(metadata)
                        if compress:
                            metadata_bytes = zlib.compress(metadata_bytes, 6)

                        f.write(struct.pack('< I', len(metadata_bytes)))
                        f.write(metadata_bytes)

                        f.seek(header_pos + 12)
                        f.write(struct.pack('< I', packet_count))

                        chksum = zlib.crc32(struct.pack(
                            '< IIII', LIGHTBIN_VERSION, creation_time, packet_count, FLAG
                        )) & 0xFFFFFFFF

                        f.seek(header_pos + 20)
                        f.write(struct.pack('< I', chksum))

                    print(f"[+] Saved {packet_count} packets to {filename}")
                    return True

                except Exception as e:
                    print(f"[-] Error saving LightBin: {e}")
                    return False

Loading from LightBin

            def load_binary(filename):
                try:
                    with open(filename, 'rb') as f:
                        header_data = f.read(24)
                        if len(header_data) != 24:
                            raise ValueError("Invalid LightBin file")

                        magic, version, created, count, flags, ck = struct.unpack('<4sIIIII', header_data)

                        if magic != LIGHTBIN_MAGIC:
                            raise ValueError(f"Invalid LightBin magic: {magic}")

                        is_compressed = bool(flags & FLAG_COMPRESSED)

                        chksum = zlib.crc32(struct.pack(
                            '< IIII', version, created, count, flags
                        )) & 0xFFFFFFFF

                        print(f"[+] Loading LightBin file...")
                        print(f"    Version: {version}")
                        print(f"    Created: {datetime.fromtimestamp(created).strftime('%Y-%m-%d %H:%M:%S')}")
                        print(f"    Packets: {count}")

                        if chksum != ck:
                            print(f"    [!] Checksum: INVALID")
                        else:
                            print(f"    [+] Checksum: Valid")

                        packets = []
                        packet_timestamps = []

                        for i in range(count):
                            pkt_header = f.read(12)
                            if len(pkt_header) != 12:
                                break

                            timestamp, size = struct.unpack('< dI', pkt_header)
                            pkt_data = f.read(size)

                            if len(pkt_data) != size:
                                break

                            if len(pkt_data) > 0:
                                first_byte = pkt_data[0]
                                if first_byte in [0x45, 0x46]:
                                    try:
                                        from scapy.layers.inet import IP as IPLayer
                                        packet = IPLayer(pkt_data)
                                    except:
                                        packet = Ether(pkt_data)
                                elif first_byte == 0x60:
                                    try:
                                        from scapy.layers.inet6 import IPv6 as IPv6Layer
                                        packet = IPv6Layer(pkt_data)
                                    except:
                                        packet = Ether(pkt_data)
                                else:
                                    try:
                                        packet = Ether(pkt_data)
                                    except:
                                        packet = Ether(pkt_data)
                            else:
                                packet = Ether(pkt_data)

                            packets.append(packet)
                            packet_timestamps.append(timestamp)

                        metadata_size_bytes = f.read(4)
                        if metadata_size_bytes:
                            metadata_size = struct.unpack('< I', metadata_size_bytes)[0]
                            metadata_bytes = f.read(metadata_size)
                            if is_compressed:
                                try:
                                    metadata_bytes = zlib.decompress(metadata_bytes)
                                except zlib.error:
                                    print("[!] Warning: Could not decompress metadata")
                            metadata = pickle.loads(metadata_bytes)
                        else:
                            metadata = {}

                        metadata['packet_timestamps'] = packet_timestamps

                        print(f"[+] Loaded {len(packets)} packets from {filename}")
                        return packets, metadata

                except FileNotFoundError:
                    print(f"[-] File not found: {filename}")
                    return None, None
                except Exception as e:
                    print(f"[-] Error loading LightBin: {e}")
                    return None, None

Detecting File Type

            def detect_file_type(filename):
                try:
                    with open(filename, 'rb') as f:
                        magic = f.read(4)
                        if magic == LIGHTBIN_MAGIC:
                            return 'lightbin'
                        elif magic[:4] == b'\xd4\xc3\xb2\xa1' or magic[:4] == b'\xa1\xb2\xc3\xd4':
                            return 'pcap'
                        elif magic[:4] == b'\x0a\x0d\x0d\x0a':
                            return 'pcapng'
                        else:
                            return 'unknown'
                except:
                    return 'unknown'

Using LightBin with LightSniff

Command-Line Usage

            # Capture and save as LightBin (with compression)
            LightSniff -i eth0 -c 100 --bin-save capture.lbn -C

            # Capture and save as LightBin (no compression)
            LightSniff -i eth0 -c 100 --bin-save capture.lbn

            # Load and display LightBin file
            LightSniff --bin-load capture.lbn -v

            # Load LightBin and save as PCAP
            LightSniff --bin-load capture.lbn -w output.pcap

            # Read PCAP and save as LightBin
            LightSniff -r input.pcap --bin-save output.lbn -C

Programmatic Usage

            from LightSniff import load_binary, save_binary
            from scapy.all import sniff

            packets = sniff(iface="eth0", count=100)

            save_binary("capture.lbn", packets, compress=True)

            loaded_packets, metadata = load_binary("capture.lbn")

            print(f"Tool: {metadata.get('tool')}")
            print(f"Packet types: {metadata.get('packet_types')}")
            print(f"Packet count: {metadata.get('packet_count')}")

            for pkt in loaded_packets:
                print(pkt.summary())

💡 When to Use LightBin

  • Large packet captures — Faster loading times
  • Automated analysis — Rich metadata for context
  • Light-Scan workflows — Native integration
  • Integrity verification — Built-in checksum

💡 When to Use PCAP

  • Interoperability — Works with Wireshark
  • Sharing — Industry standard format
  • Legacy tools — Older tools only support PCAP
  • Smallest file size — PCAP is slightly smaller

Compatibility

LightBin is designed to be backward-compatible. Future versions will continue to support older LightBin files. The version header allows parsers to handle different format versions gracefully.
